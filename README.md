# meetup-ticket-selling
Adding online ticket selling to events on meetup.com website

# HOW to run
- terminal in new tab

    > npm i

    > webpack -wd

- terminal in new tab

    > export MONGO_URL=mongodb://<dbuser>:<dbpassword>@ds055505.mongolab.com:55505/ticket_event
    
    > nodemon
    
- navigate to localhost:3000
